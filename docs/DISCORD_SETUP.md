# 💬 디스코드 설정 가이드

## 1. 디스코드 서버 생성

디스코드에서 새 서버를 생성하고 다음 채널을 만드세요:

| 채널 | 용도 |
|------|------|
| `#매매신호` | 매수/매도 알림 |
| `#일일리포트` | 일일 성과 요약 |
| `#에러로그` | 에러/경고 알림 |
| `#시스템상태` | 봇 상태/하트비트 |

## 2. Webhook URL 생성

각 채널에 대해:

1. 채널 설정 → **연동** → **웹훅**
2. **새 웹훅** 클릭
3. 이름 설정 (예: "Trading Bot")
4. **웹훅 URL 복사**

## 3. .env 파일 설정

```
DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/.../...
DISCORD_WEBHOOK_ERROR_URL=https://discord.com/api/webhooks/.../...
DISCORD_WEBHOOK_REPORT_URL=https://discord.com/api/webhooks/.../...
```

## 4. 연결 테스트

```bash
python scripts/setup_discord.py
```

## 5. 알림 종류

| 알림 | 색상 | 설명 |
|------|------|------|
| 매수 실행 | 🟢 초록 | 페어, 진입가, 수량, 목표가, 손절가, 점수 |
| 매도 (익절) | 🔵 파란 | 페어, 진입/청산가, 손익, 보유시간 |
| 매도 (손절) | 🔴 빨간 | 동일 |
| 에러/경고 | 🟠 주황 | 에러 메시지 |
| 하트비트 | ⚪ 회색 | 상태, 포지션 수, 일일 손익 |
| 비상 정지 | 🔴 빨간 | 즉시 확인 필요 |
| 초기화 완료 | ⚙️ 시스템 | 리셋 상세 내역 및 시작 잔고 |
| 미관리 청산 | ⚠️ 주황 | 유령 포지션 감지 및 자동 청산 결과 |

## 6. Rate Limit

디스코드 Webhook Rate Limit:
- 초당 5회
- 분당 30회

봇이 자동으로 Rate Limit을 준수합니다.
